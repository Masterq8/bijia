# 遥感数据检索与可视化系统前端交接文档

## 1. 项目概述

本项目是一个遥感数据检索与可视化系统，主要功能包括：
- 遥感数据检索（按类型、区域、时间）
- 地图可视化展示
- 卫星数据详情查看
- 专题图制作
- 数据上传
- 智能问答系统

## 2. 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue | 3.x | 前端框架 |
| Element Plus | 最新版 | UI组件库 |
| Leaflet | 最新版 | 地图可视化 |
| Vuex | 4.x | 状态管理 |
| Axios | 最新版 | HTTP请求 |
| SCSS | 最新版 | 样式管理 |

## 4. 核心页面结构

### 4.1 主页面 (Home.vue)

主页面分为左侧控制面板和右侧地图展示区域：

#### 左侧控制面板

1. **数据类型选择**
   - 使用Element Plus的Tree组件
   - 支持多选，选择后会筛选相应类型的卫星数据

2. **检索区域选择**
   - 支持两种方式：
     - 省份城市选择（使用Element Plus的Select组件）
     - 地图绘制多边形区域（使用Leaflet的绘图功能）

3. **采集时间选择**
   - 使用Element Plus的DatePicker组件（范围选择）
   - 提供快捷时间标签（近三天、近一周、近一月等）

4. **其他选项**
   - 保持查询结果（复选框）
   - 上传我的数据（按钮）

5. **操作按钮**
   - 快速上手
   - 检索

#### 右侧地图展示区域

1. **地图组件**
   - 使用Leaflet实现
   - 支持缩放、平移等基本操作
   - 支持绘制多边形区域

2. **卫星数据展示**
   - 在地图上显示卫星数据覆盖区域（多边形）
   - 支持点击多边形查看详情
   - 支持双击多边形显示/隐藏卫星影像

3. **智能问答系统**
   - 右下角机器人图标，点击打开问答弹窗
   - 支持用户输入问题，返回相关回答

### 4.2 数据详情弹窗

- 显示卫星数据的详细信息
- 包括卫星类型、传感器类型、采集时间、经纬度范围等
- 提供下载、收藏、处理分析等操作按钮

### 4.3 专题图制作弹窗

- 输入专题图标题
- 选择专题图类型
- 输入备注信息
- 预览专题图效果
- 提供制作按钮

### 4.4 数据上传弹窗

- 选择采集时间
- 选择所属区域（省份、城市）
- 上传影像文件（支持zip、tar.gz、rar格式）
- 提供解压并上传按钮

## 5. 核心功能实现

### 5.1 地图功能 (Map.vue)

1. **初始化地图**
   - 使用Leaflet创建地图实例
   - 设置地图中心和缩放级别
   - 添加高德地图底图
   - 添加机器人控件（智能问答系统入口）

2. **绘制多边形**
   - 使用Leaflet的绘图插件
   - 支持在地图上绘制多边形区域
   - 绘制完成后获取多边形的经纬度坐标

3. **添加多边形和影像**
   - 根据卫星数据的经纬度范围创建多边形
   - 支持在多边形位置添加卫星影像
   - 支持旋转影像以匹配实际地理方向

4. **地图交互**
   - 支持点击多边形查看详情
   - 支持双击多边形显示/隐藏影像
   - 支持右键多边形打开详情弹窗

### 5.2 数据检索功能 (Home.vue)

1. **构建查询参数**
   - 收集用户选择的数据类型、区域、时间等参数
   - 转换为API接口所需的格式

2. **发送检索请求**
   - 调用listSatellite1接口获取卫星数据
   - 处理返回的数据，更新卫星数据列表

3. **展示检索结果**
   - 在左侧面板显示卫星数据列表
   - 在地图上显示卫星数据覆盖区域
   - 支持分页显示

### 5.3 数据详情功能

1. **查看详情**
   - 点击列表项或地图多边形打开详情弹窗
   - 显示卫星数据的详细信息

2. **下载影像**
   - 点击下载按钮下载卫星影像
   - 需要用户登录

3. **收藏影像**
   - 点击收藏按钮将影像添加到用户收藏
   - 需要用户登录

4. **处理分析**
   - 点击处理分析按钮对影像进行处理
   - 需要用户登录

### 5.4 专题图制作功能

1. **选择专题图类型**
   - 从下拉列表选择专题图类型
   - 实时预览专题图效果

2. **提交制作请求**
   - 点击制作按钮提交专题图制作请求
   - 调用addMake接口

### 5.5 数据上传功能

1. **选择文件**
   - 使用Element Plus的Upload组件
   - 支持拖拽上传

2. **提交上传请求**
   - 点击解压并上传按钮
   - 调用文件上传接口

### 5.6 智能问答系统

1. **打开问答弹窗**
   - 点击地图右下角的机器人图标

2. **发送问题**
   - 在输入框中输入问题
   - 点击发送按钮

3. **获取回答**
   - 调用getByContent接口获取回答
   - 在弹窗中显示问答历史

## 6. 核心API接口

| 接口名称 | 用途 | 调用位置 |
|---------|------|----------|
| listSatellite1 | 检索卫星数据 | Home.vue:search() |
| getByContent | 智能问答 | Map.vue:backRequest() |
| imageTypeGet | 获取专题图类型 | Home.vue:onMounted() |
| addMake | 制作专题图 | Home.vue:makeButtton() |
| addCollectionInfo | 收藏影像 | Home.vue:collectImage() |
| listInfo | 获取收藏列表 | Home.vue:getCollection() |
| addInfo | 添加处理分析任务 | Home.vue:ProcessingAnalysis() |
| addDateInfo | 加入我的数据 | Home.vue:handleAddDate() |

## 9. 注意事项

1. **地图功能**
   - 使用Leaflet库实现地图功能，需要引入相关依赖
   - 地图底图使用高德地图服务
   - 影像叠加需要使用leaflet-imageoverlay-rotated插件

2. **数据检索**
   - 检索条件需要至少选择数据类型和区域
   - 时间范围默认为当前月
   - 检索结果会在地图上显示对应的多边形区域

3. **用户权限**
   - 下载、收藏、处理分析功能需要用户登录
   - 未登录用户会禁用这些功能

4. **文件上传**
   - 支持的文件格式：zip、tar.gz、rar
   - 文件大小限制：最大5G
   - 上传后会自动解压并处理

5. **智能问答**
   - 基于后端API实现
   - 点击地图右下角机器人图标打开
